Web responsive

Cuando utilizamos el termino "responsive"(adaptable), nos referimos principalmente a "responsive design" (diseño web adaptable). Esto significa hacer que un sitio web sea accesible y adaptable en todos los devices: tabletas, smartphones, etc.
